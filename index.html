Colors Of Pi

This is a more "artsy" project that depicts the hexadecimal colors that pi makes when you split pi into sixes. Using python, I took a text file with the digits of pi, then split them up every six characters and saved them into an html file as div tags. For example: 3.14159 becomes #314159, which is a shade of blue, and 265358 becomes #262358 which is a shade of greyish-green. Check out the site. It is actually pretty cool, especially if you like math. If you are a true nerd, then make it your desktop screen saver! Also if you're interested, the source code for the python file that parses pi and creates this page will be available online soon.

Random Walks

This is an art project that bears resemblance to a screen saver. A random walk is a mathematical concept that describes natural movements, like the path a dandelion seed takes in the wind. That means taking a step at a fixed size in one direction, then turning to a random direction and taking another step of the same size. This page takes that math concept and turns it into art.
